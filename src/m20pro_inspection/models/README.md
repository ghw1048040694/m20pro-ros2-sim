# Inspection Models

Place your converted RKNN model here for the default launch path:

```text
inspection.rknn
```

Optional class names can be stored one per line:

```text
classes.txt
```

Model artifacts such as `*.pt`, `*.onnx`, and `*.rknn` are ignored by Git.

Dataset/training reference:

```text
https://github.com/liu-big/power_station_safety.git
```
